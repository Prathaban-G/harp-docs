Step 3 - Install Dependencies

Update Linux​

Update OS​

This command (sudo apt-get update) updates the package lists for upgrades and installs any available updates to the operating system.

sudo apt-get update

Install Build Essentials​

Installing build-essential (sudo apt-get install -y build-essential) provides a set of essential tools required for compiling and building software from source code on Linux systems. This includes compilers, make tools, and libraries necessary for building many applications.

sudo apt-get install -y build-essential

Install Git​

Git is a version control system used for tracking changes in source code during software development. Installing Git (sudo apt-get install -y git) allows users to clone, manage, and collaborate on projects efficiently.

sudo apt-get install -y git

Install OpenVPN​

OpenVPN is an open-source virtual private network (VPN) software that enables secure communication over the internet. Installing OpenVPN (sudo apt-get install -y openvpn) allows users to set up VPN connections for secure data transmission.

sudo apt-get install -y openvpn

Install PIP and Virtual Environment for Python​

PIP is a package management system used to install and manage software packages written in Python. Additionally, setting up a virtual environment (python3-venv) allows users to create isolated Python environments for different projects. Installing PIP and Python virtual environment (sudo apt-get install -y python3-pip python3-venv) is essential for managing Python packages and dependencies.

sudo apt-get install -y python3-pip python3-venv

Install IP Tables​

IPTables is a user-space utility program that allows configuring and managing the kernel's packet filtering rules for network traffic. Installing IPTables (sudo apt-get install -y iptables) provides users with tools for firewall configuration and network traffic control.

sudo apt-get install -y iptables

Install IP Tables - Persistent​

Installing iptables-persistent (sudo apt-get install -y iptables-persistent) ensures that IPTables rules persist across system reboots. This package automatically saves and loads firewall rules during system startup and shutdown, maintaining consistent firewall configurations.

sudo apt-get install -y iptables-persistent

Install Minicom​

Minicom is a terminal emulation program that allows users to communicate with devices such as modems, routers, and serial consoles. Installing Minicom (sudo apt-get install -y minicom) provides a user-friendly interface for serial communication, facilitating tasks such as debugging and configuring hardware devices.

sudo apt-get install -y minicom
