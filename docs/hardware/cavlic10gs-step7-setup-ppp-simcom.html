Step 7 - wwan on SIMCOM

Note

This guide is only for Devices connected with SIMCOM 7600E Modem.

Overview​

The SIMCOM 7600E GSM modem module is connected to PCIe, which by default creates the usb0 interface for accessing the internet. However, instead of relying on the usb0 interface, we prefer to use wwan for establishing a more reliable and stable internet connection. PPP offers better performance and flexibility compared to the default interface, making it the preferred choice for internet connectivity on the SIMCOM 7600E modem.

Why It's Needed:​

Creating and configuring network connections via nmcli is essential for network administrators and users who prefer command-line interfaces for managing network settings. It offers flexibility and fine-grained control over network configurations, making it suitable for various networking tasks.

Steps To Create a Connection :​

Step 1: Create a New Network Connection:​ 

sudo  nmcli con add type gsm con-name "harp-gsm"

Step 2: Edit the Connection Settings:​ 

sudo  nmcli con edit harp-gsm

Step 3: View Connection Status:​

 print

Step 4: Modify Connection Settings:​

Configure interface Connection

set connection.interface-name cdc-wdm0

Set the Access Point Name (APN)

set gsm.apn [APN NAME]

Set default network

set ipv4.never-default no

Save the settings

save

Exit nmcli

quit

Conclusion​

Using nmcli simplifies the process of creating and configuring network connections on Linux systems. It provides a straightforward way to manage network settings efficiently, especially in scenarios where a command-line interface is preferred or necessary. By following the outlined steps, you can create a GSM-type network connection, configure its settings, and set the default network interface as per your requirements.
